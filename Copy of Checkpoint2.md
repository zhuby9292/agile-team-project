# Agile Web Development Group Project

## Overview

Our application is a course selection and timetable planning system designed to help students organise their semester efficiently.

The system allows users to select their degree and browse available courses, including details such as course codes, credit points, and scheduling information. Students can then add courses to their plan and generate a structured timetable based on their selections.

The application also aims to improve decision-making by allowing users to view other users' timetables. This helps students explore different course combinations and optimise their study plans.

In addition, the system includes administrative features where teachers or administrators can monitor course enrollment statistics. This provides insights into course demand and helps support better planning and management.

Overall, the system focuses on providing a simple, intuitive, and user-friendly interface that supports efficient course planning and collaboration between users.

## User Roles

This system will support different types of users:

- Students: can register, log in, select courses, and generate timetables.
- Teachers/Admin: can view enrollment statistics and monitor course selections.

To prevent misuse, teacher/admin accounts will require additional verification, such as admin approval or a special registration code. 

## System Workflow

The system will follow a simple workflow:
1. Users register and log in to the system.
2. Students select their degree and view available courses.
3. Students add course details such as course code, credit points, and time.
4. The system generates a timetable based on selected courses.
5. Students can view and share their timetable.
6. Admin users can view enrolment statistics and monitor course selections.

## User stories

- As a student, I want to sign up with my details so that I can create a personal account.
- As a student, I want to log in securely so that I can access my selected courses.
- As a student, I want to select my degree so that I can view relevant courses for my program.
- As a student, I want to add course details such as course code, credit points, and time, so that I can build my timetable.
- As a student, I want to remove or update selected courses so that I can adjust my schedule.
- As a student, I want to generate a timetable automatically, so that I can see a structured weekly plan.
- As a student, I want to view my timetable in a calendar format, so that I can easily understand my schedule.
- As a student, I want to share my timetable with others, so that I can get feedback from classmates.
- As a student, I want to view other students’ schedules so that I can get ideas for my own course selection.
- As a teacher, I want to view the list of students enrolled in my course, so that I can understand class size and manage my teaching.
- As an admin, I want to approve teacher registrations so that only authorised users can access teacher functionalities.
- As an admin, I want to manage user roles so that I can control access to different parts of the system.  
-
- As a student, I want to view a list of all the courses I have added, so that I can manage my possible unit choices easily.
- As a teacher, I want to publish notes or simple advice for a course so that students can make better course selections.
- As a teacher, I want to see whether a course has time clashes with other popular units, so that I can identify scheduling problems.
- As an admin, I want to add, update, or remove course records so that the system always contains current course information.
